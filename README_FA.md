Ш§ЫҢЩҶ ЩҫШұЩҲЪҳЩҮШҢ ЫҢЪ© ЩҶШіШ®ЩҮвҖҢЫҢ Ш§ШұШӘЩӮШ§ЫҢШ§ЩҒШӘЩҮ Ш§ШІ ЩҫЩҶЩ„ **[3X-UI](https://github.com/MHSanaei/3x-ui)** (ЩҶШіШ®ЩҮвҖҢЫҢ 2.9.3) ЩҮШіШӘШҙ.  ЩҮШҜЩҒ Ш§ЫҢЩҶ ЩҫШұЩҲЪҳЩҮ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ШұШҜЩҶ ЩҫШұЩҲШӘЪ©Щ„ ЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒ ЩҲ ШұШ§ЩҮ Ш§ЩҶШҜШ§ШІЫҢ ШЁШөЩҲШұШӘ ЫҢЪ© ЩҫЩҶЩ„ Ш¬Ш§Щ…Ш№ЩҮ ШЁШ§ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ ЩӮШ§ШЁЩ„ЫҢШӘ ЩҮШ§ЫҢ **Xray-core**  ЩҮШіШӘШҙ

## ЩҫШұЩҲШӘЪ©Щ„вҖҢЩҮШ§ЫҢ Ш¬ШҜЫҢШҜ

## Ш§Щ…Ъ©Ш§ЩҶШ§ШӘ Ш¬ШҜЫҢШҜ

- ЩӮШ§ШЁЩ„ЫҢШӘ **Client to Client** ШӯШӘЫҢ ШЁШөЩҲШұШӘ **Cross Inbound** (Ш§ШӘШөШ§Щ„ ШҜШ§Ш®Щ„ЫҢ Ъ©Ш§ШұШЁШұ L2TP ШЁЩҮ Ъ©Ш§ШұШЁШұ OpenVPN)
- Ш§Ш¶Ш§ЩҒЩҮвҖҢШҙШҜЩҶ **Encryption** ЩҮШ§ЫҢ **AES-256-GCM** ЩҲ **AES-128-GCM** ШЁЩҮ ЩҫШұЩҲШӘЪ©Щ„ **Shadowsocks**
- ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ **XHTTP Object** ШҜШұ **Outbound**
- Ш§ШіЪ©ШұЫҢЩҫШӘ ЩҶШөШЁ Ш®ЩҲШҜЪ©Ш§Шұ **[WARP-CLI](https://github.com/Sir-MmD/warp-cli)** (ЩҶШіШ®ЩҮвҖҢЫҢ ШұШіЩ…ЫҢ Cloudflare)
- ЩҮШіШӘЩҮвҖҢЫҢ [**Xray-core** ЩҫЪҶвҖҢШҙШҜЩҮ](https://github.com/Sir-MmD/Xray-core) ШЁШұШ§ЫҢ ШұЩҒШ№ Ш®Ш·Ш§ЫҢ В«Unsupported CipherВ» ШҜШұ ЩҫШұЩҲШӘЪ©Щ„ **Shadowsocks**
- ШЁШ§ЩҶШҜЩ„вҖҢШҙШҜЩҶ ЩҮЩ…ЩҮвҖҢЫҢ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ (GeofileШҢ Xray-core ЩҲ ЩҮШіШӘЩҮвҖҢЩҮШ§ЫҢ Backend) ШҜШ§Ш®Щ„ ЫҢЪ© ЩҒШ§ЫҢЩ„ ШЁШ§ЫҢЩҶШұЫҢЩҗ ЩҲШ§ШӯШҜ
- Ш®ШұЩҲШ¬ЫҢ ЪҜШұЩҒШӘЩҶ Щ„ЫҢЩҶЪ© Ш§Ъ©Ш§ЩҶШӘ ЩҮШ§ ШЁШөЩҲШұШӘ **TXT** ЩҲ **PDF**
- ЩӮШ§ШЁЩ„ЫҢШӘ **Freez** Ъ©ШұШҜЩҶ Ш§Ъ©Ш§ЩҶШӘ ЩҮШ§ШӘ
- Ш§Ш¶Ш§ЩҒЩҮ ШҙШҜЩҶ **checkbox** ШЁЩҮ Ъ©Щ„Ш§ЫҢЩҶШӘ ЩҲ Inbound ЩҮШ§
- ЩӮШ§ШЁЩ„ЫҢШӘ **Bulk Operation**: 

    * ШӘШәЫҢЫҢШұ ЪҜШұЩҲЩҮЫҢ ШӯШ¬Щ… Ш§Ъ©Ш§ЩҶШӘ ЩҮШ§
    * ШӘШәЫҢЫҢШұ ЪҜШұЩҲЩҮЫҢ ШұЩҲШІ Ш§Ъ©Ш§ЩҶШӘ ЩҮШ§
    * ЩҒШ№Ш§Щ„ ШіШ§ШІЫҢ/ШәЫҢШұ ЩҒШ№Ш§Щ„ ШіШ§ШІЫҢ ЪҜШұЩҲЩҮЫҢ Ш§Ъ©Ш§ЩҶШӘ ЩҮШ§
    * ШӯШ°ЩҒ ЪҜШұЩҲЩҮЫҢ Ш§Ъ©Ш§ЩҶШӘ ЩҮШ§
    * ШӯШ°ЩҒ ЪҜШұЩҲЩҮЫҢ Inbound ЩҮШ§
    * ЩӮШ§ШЁЩ„ЫҢШӘ Freez/Un-Freez Ъ©ШұШҜЩҶ ЪҜШұЩҲЩҮЫҢ Ш§Ъ©Ш§ЩҶШӘ ЩҮШ§

## ШӘШіШӘ E2E

![ШӘШіШӘ E2E](https://raw.githubusercontent.com/Sir-MmD/vpn-ui/refs/heads/main/media/test_unit.png)

ЫҢЪ© ШӘШіШӘ **E2E** Ъ©Ш§Щ…Щ„ ШЁШ§ Python ШҜШ§Ш®Щ„ ЩҒЩҲЩ„ШҜШұ `test_unit` ШЁШұШ§ЫҢ Ш§ЫҢЩҶ ЩҫШұЩҲЪҳЩҮ Ш·ШұШ§ШӯЫҢ ШҙШҜЩҮ Ъ©ЩҮ Щ…ЫҢвҖҢШӘЩҲЩҶЫҢШҜ Ш§ШІШҙ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ. Щ…ШұШ§ШӯЩ„Шҙ Ш§ЫҢЩҶвҖҢШ·ЩҲШұЫҢЩҮ:

1. ЩҲШ§ШұШҜ ЩҒЩҲЩ„ШҜШұ `test_unit` ШЁШҙЫҢШҜ ЩҲ ШӘЩҶШёЫҢЩ…Ш§ШӘ ШҜЩ„Ш®ЩҲШ§ЩҮвҖҢШӘЩҲЩҶ ШұЩҲ ШӘЩҲЫҢ `config.toml` ЩҲШ§ШұШҜ Ъ©ЩҶЫҢШҜ.
2. Ш§ШіЪ©ШұЫҢЩҫШӘ `setup.sh` ШұЩҲ Ш§Ш¬ШұШ§ Ъ©ЩҶЫҢШҜ.
3. ЩҒШ§ЫҢЩ„ ШЁШ§ЫҢЩҶШұЫҢЩҗ Ъ©Ш§Щ…ЩҫШ§ЫҢЩ„вҖҢШҙШҜЩҮ ШұЩҲ ШҜШ§Ш®Щ„ ЩҒЩҲЩ„ШҜШұ `test_subject` ЩӮШұШ§Шұ ШЁШҜЫҢШҜ.
4. `run.sh` ШұЩҲ ШЁШ§ ШҜШіШӘШұШіЫҢ `sudo` Ш§Ш¬ШұШ§ Ъ©ЩҶЫҢШҜ.

> [!IMPORTANT]
> ШӘШіШӘ Ъ©Ш§Щ…Щ„ E2E ШЁЩҮвҖҢШҙШҜШӘ ШІЩ…Ш§ЩҶвҖҢШЁШұЩҮШӣ Ш§ЪҜЩҮ ЩҒЩӮШ· ЫҢЩҮ ШӘШәЫҢЫҢШұ Ъ©ЩҲЪҶЫҢЪ© ШӘЩҲЫҢ ЩҫШұЩҲЪҳЩҮ ШҜШ§ШҜЫҢШҜШҢ ШЁЩҮШӘШұЩҮ ШЁШ§ ШіЩҲЫҢЪҶ `--tests` ЩҒЩӮШ· ЩҮЩ…ЩҲЩҶ ШЁШ®Шҙ ШұЩҲ ШӘШіШӘ Ъ©ЩҶЫҢШҜ:

| Test ID | Description |
| :--- | :--- |
| `core-init` | provision kernel modules + packages + xray core |
| `server-setup` | create inbounds + accounts + source-IP routing rules |
| `openvpn` | connect variants + checks + peer reachability (OpenVPN) |
| `l2tp` | connect variants + checks + peer reachability (L2TP/IPsec) |
| `pptp` | connect variants + checks + peer reachability (PPTP) |
| `openconnect` | connect variants + checks + peer reachability + same-NAT user-limit (OpenConnect/ocserv) |
| `sstp` | connect variants + checks + peer reachability (SSTP/accel-ppp, PPP-over-TLS) |
| `ikev2` | connect + checks + peer reachability (IKEv2/IPsec, strongSwan charon; eap-mschapv2 + psk + eap-tls) |
| `wg-c` | connect + checks + peer reachability + per-account usage/termination (WireGuard C, in-kernel wgctrl, gateway /29, + preshared-key mode) |
| `mtproto` | alias: runs every MTProto phase below (MTProto Proxy, telemt) |
| `mtproto-classic` | handshake + relay to a real Telegram DC + wrong-secret control + usage (obfuscated2) |
| `mtproto-secure` | same, "dd" random-padding secret |
| `mtproto-tls` | same + FakeTLS ServerHello HMAC verified, "ee" secret |
| `mtproto-toggle` | editing an account's modes takes effect on the RUNNING daemon (no restart) |
| `bulk-ops` | bulk client add/sub/enable/disable + TXT/PDF export via API |
| `backup-restore` | DB export + import round-trip |
| `warp-socks` | Cloudflare warp-cli SOCKS install + egress |
| `random-cfg` | `--random` switch: randomize port + creds + webpath, then restore |
| `systemd` | `--systemd` switch: install + run the panel as a systemd unit |
| `uninstall` | `--uninstall` switch: install everything, tear down, assert clean host |
| `export-js` | host-side Node TXT/PDF export test (no VM) |
